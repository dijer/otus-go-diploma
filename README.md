# Превьювер изображений

Дипломмный проект: сервис для создания изображения с новыми размерами на основе имеющегося изображения

## Installation
Установит нужные зависимости
`make install`

## run dev mode
Запуск в дев-среде
`make dev`

## build
сборка сервиса
`make build`

## clean build file
удаляет сборку
`make clean`

## start service
Запуск сервиса
`make run`

## stop service
Останавливает сервис
`make stop`

## integration tests
Запуск интеграционных тестов
`make integration`

## other tests
Запуск остальных тестов
`make test`

## linting
Запустить линтинг
`make lint`
